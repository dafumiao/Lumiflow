# Lumiflow 应用图标设计说明

## 设计元素
- **蓝色渐变背景**：代表专业级的摄影光效环境
- **光芒射线**：象征应用的核心功能——提供各种颜色的补光效果
- **相机镜头**：表示应用的摄影功能
- **光圈结构**：中央的多层环形设计代表可调节的亮度和饱和度控制
- **金色光线**：强调补光的温暖效果和专业感

## 颜色选择
- 主色调使用蓝色：给人科技感和专业性
- 金黄色光线：提供温暖的视觉对比，象征灯光
- 深蓝色相机元素：增强可识别性

## 视觉风格
整体设计简洁明了，符合现代iOS应用设计语言，同时具有足够的辨识度。图标在小尺寸下也能保持清晰可辨识。
